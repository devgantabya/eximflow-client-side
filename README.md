# EXIM FLOW

**Live URL:** [https://eximflow-9f432.web.app/](https://eximflow-9f432.web.app/)

**A modern web platform where users can manage exports, browse global products, and import any product into their personal "My Imports" section with one click.**

---

## Overview

**EXIM FLOW** is a products export-import hub where users can:

- Post items for sale (`Products`)
- Import items (`Imports`)
- Negotiate quantities through importing
- Export items (`Exports`)
- Take actions (`Post`, `Update`, `Remove`)

The platform provides a seamless experience for buyers and sellers in global trade.

---

## Features

- User registration and authentication (Firebase Auth)
- Browse and search products
- Import/export products
- View personal imports in "My Imports" section
- Responsive UI for desktop and mobile
- Smooth navigation with React Router
- Loader and Suspense for async data fetching
